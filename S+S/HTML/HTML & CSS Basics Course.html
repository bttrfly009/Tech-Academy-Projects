Setting up the development environment

☑ Please hang on to all tutorial files as we will be revisiting them in future tutorials.
☑ Each tutorial will have downloadable files, so you can follow along.
☑ Please follow any instructions that explain which directories the files need to go inside of.

The first thing that we must do is setup our development environment to organize where and how we will be storing our files. To begin, we need to create a new folder on the desktop. Inside the new folder, create a text file and rename it index.html. We will also need to create two additional folders; naming one: "css", and the other: "images".

The css folder will contain our future Cascading Style Sheets (CSS) for stylizing our web pages, making them all beautiful. The images folder will be our location for storing all of our project images.

Now that we have our working directory all setup, we need to right click the index.html file that we have just created and select, "Edit with Notepad++". This will open our new HTML file into Notepad++. Notepad is basically a glorified text editor and not a true Integrated Development Environment, (IDE) but it is a capable and free application that offers basic IDE-like features that we will use as we build our web site. Features such as: colored syntax, code indentation, split-screen views, numbered code-lines, tag highlighting, and a pseudo-intellisense (auto text completion). But in a nut-shell, an IDE is intended to make developers' lives much easier as they code.
